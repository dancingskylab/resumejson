## 0.1.0

- Initial release of `package:resumejson`.
- Implements ResumeJSON Specification v0.1.0.
- 15 immutable data classes with `const` constructors, `copyWith()`,
  `fromJson()` / `toJson()`, and value equality.
- 10 Dart enums for all constrained string fields.
- Full JSON-LD serialization with `@context`, `@type`, and `resumejson:`
  namespace keys.
- Zero external dependencies.
